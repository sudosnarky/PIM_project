Help & FAQs

What is PARA InfoSystem?

PARA InfoSystem is a simple, powerful tool to organize your notes, tasks, and knowledge using the PARA method: Projects, Areas, Resources, Archives. It is designed to be so easy that anyone can use it!

Getting Started

  1. Register for an account on the login page.
  2. After registering or logging in, you'll see an onboarding page explaining PARA and how to use the app.
  3. Go to your Dashboard to see, create, and manage your notes (called "particles").

FAQs

What is PARA?

PARA stands for Projects, Areas, Resources, and Archives. It's a method to organize everything in your life and work. Watch the onboarding video for a quick explanation!

How do I create a new note/particle?

Click the Create Particle button on the dashboard. Fill in the title, content (you can use Markdown!), and add tags if you want. Click Save to add it.

How do I edit or view a note?

On the dashboard, each note has View and Edit buttons. Click View to see the full note, or Edit to make changes.

How do I move a note between PARA sections?

On the dashboard, use the Move dropdown next to each note to change its section (Projects, Areas, Resources, Archives). To archive a note, click the Archive button.

What are tags and how do I use them?

Tags are keywords you add to notes for easy searching and grouping. Add tags separated by commas when editing or creating a note. Click a tag to see all notes with that tag!

How do I search for notes?

Use the search bar at the top of the dashboard. Type any word or tag to filter your notes instantly.

Can I use formatting in my notes?

Yes! You can use Markdown for bold, italics, code, links, lists, and more. Try it out in the note editor.

Is my data private?

Yes. Only you can see your notes after logging in.

What if I forget my password?

Currently, there is no password reset. Please contact the app owner if you need help.

Can I use PARA InfoSystem on my phone or tablet?

Yes! The app works on any device with a modern web browser.

How do I log out?

Click the Logout button at the top right of the dashboard.

How do I get back to this Help page?

Bookmark this page or look for a Help or FAQs link in the app navigation.

Tips for Everyone (Even Babies!)

  • Click big buttons. They do what they say!
  • Can't find something? Use the search bar.
  • Want to organize? Use tags and PARA sections.
  • Need help? Read this page or ask a grown-up.
  • Don't worry about making mistakes. You can always edit or move your notes.
  • Have fun and be creative!
