How to Use This App

  • Projects: Short-term efforts with a clear outcome. Create, edit, and manage your projects as particles.
  • Areas: Ongoing responsibilities. Use this section for recurring tasks and notes.
  • Resources: Reference material, knowledge, and learning. Store useful info here.
  • Archives: Completed or inactive notes. Move finished items here for future reference.
  • Tags: Add hashtags to notes for easy filtering. Click a tag to see all related notes.
  • Markdown: Write notes in Markdown for formatting (bold, italics, code, links, etc.).
  • Search: Use the dashboard search bar to quickly find notes by title or content.
  • Security: All your notes are private to your account.
